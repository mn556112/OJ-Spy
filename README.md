# 🧮 Sejong OJ 자동 채점 프로그램 (OJ-Spy)

Sejong Online Judge(EX-OJ)의 채점 결과를 자동으로 수집하고  
학생별 점수를 계산하여 엑셀 파일로 저장하는 Python 기반 GUI 프로그램입니다.

Python + Tkinter로 제작되었으며, 문제별 점수를 자동 수집하여  
총점 및 등급을 계산하고 Excel 파일로 저장할 수 있습니다.

---

## ✨ 주요 기능

### 🔹 OJ 자동 로그인
- OJ ID / PW 입력만 하면 자동 로그인
- 303 Redirect + Location 검사 기반 로그인 성공 판별

### 🔹 문제 목록 자동 수집
- 문제 리스트 URL 입력 → 문제명 / 문제ID / status URL 자동 파싱

### 🔹 점수 자동 계산
- 학번 여러 개 자동 반복 채점
- 문제별 최고 점수 자동 수집
- **그룹 점수 계산 규칙 적용**
  - 1-1 문제: 50% 반영
  - 1-2 문제 존재 시: 1-1 무시, 100% 반영
  - 1-1 절반 vs 1-2 점수 비교 → 더 큰 점수 반영

### 🔹 등급 계산 기능 (선택)
- A / B / C / D / F 비율 입력 가능
- 비율 기반 자동 등급 배정
- 상위 절반 + (A+, B+, C+) / 하위 절반 0 (A0, B0, C0)
- **0점인 학생은 무조건 F 처리**

### 🔹 결과 저장
- 저장 위치 직접 선택 가능
- Excel 파일 (`scores.xlsx`) 자동 생성

### 🔹 진행바 + ETA 지원
- 실시간 진행률 & 예상 완료 시간 표시
- 중도 취소 기능 제공

---
## 🥹사용방법
- OJ 로그인 정보 입력

- 문제 리스트 URL 입력

- 학번 목록 입력

- (선택) 등급 비율 설정

- 저장 폴더 선택

- 시작 버튼 클릭 → 자동 채점

- 엑셀 파일 생성 및 확인

## ⚠️ 주의사항
- 해당 프로그램을 악의적으로 사용하여 발생하는 모든 불상사(계정 차단, 점수 오류, 서버 측 제재 등)에 대한 책임은 사용자 본인에게 있으며, 개발자는 어떠한 책임도 지지 않습니다.  
- 프로그램은 교육적 목적 및 편의를 위해 제작되었으며, 타인의 계정 정보를 무단으로 사용해서는 안 됩니다.  
- EX-OJ 서버 구조가 변경될 경우 프로그램이 정상 작동하지 않을 수 있으며, 이때 발생하는 문제에 대해 개발자는 보장을 제공하지 않습니다.  
- 자동 크롤링 기능은 서버에 부담을 줄 수 있으므로, 과도한 요청을 보내지 않도록 주의하세요.
